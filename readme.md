# 随机笑话页面
[![v4zBY4.md.png](https://s1.ax1x.com/2022/08/31/v4zBY4.md.png)](https://imgse.com/i/v4zBY4)

使用了以下特性搭载运行环境：  
Babel Loader  
Sass Loader With Dart Sass  
HTMLWebpackPlugin  
WebpackDevServer  
Asset Resource Loader  
Source Maps  
Caching

## 使用

开发

```
npm run dev
```

生产

```
npm run build
```
